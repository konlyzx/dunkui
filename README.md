## ğŸ“¦ Installation

DunkUI is designed to be copy-pasted into your project.

1.  **Browse the components** on our documentation site.
2.  **Copy the code** for the component you want.
3.  **Paste it** into your SvelteKit project.
4.  **Install dependencies** (if any) listed on the component page.

Example:

```bash
npm install ogl gsap
```
